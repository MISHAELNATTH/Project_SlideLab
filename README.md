## Info :
Il faut installer Node.js et Git afin d'utiliser le projet et de le partager.
Ensuite, nous utilisons ces bibliotèque à installer :
npm install --save-dev gh-pages
npm install react react-dom
npm install -D vite @vitejs/plugin-react tailwindcss @tailwindcss/vite

Pour en dépot en ligne du site il faut :
To init a git on pages :
git init
git add .
git commit -m "Initial commit"

git branch -M main
git remote add origin https://github.com/Florian-Glay/Projet-Informatique-S6-GR4
git push -u origin main

TO DO EVERY NEW VERSION : npm run deploy
